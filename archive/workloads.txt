This file contains the details of different workloads generated as part of the experiments. 


1. init_graph: This workload is generated by running the script in "scripts/init_graph.py". This will initialize a social graph based on Reed98 Facebook Networks, with 962 users and 18.8K social graph edges.

    python3 scripts/init_graph.py


2. compose_post: This workload is generated using wrk2. The command that generated the workload is:

    ./wrk -D exp -t 4 -c 100 -d 120 -L -s ./scripts/social-network/compose-post.lua http://localhost:8080/wrk2-api/post/compose -R 1000


3. read_home_timeline: This workload is generated using wrks2. The command that generated the workload is: 

    ./wrk -D exp -t 4 -c 100 -d 120 -L -s ./scripts/social-network/read-home-timeline.lua http://localhost:8080/wrk2-api/home-timeline/read -R 1000


4. read_user_timeline: This workload is generated using wrks2. The command that generated the workload is: 

    ./wrk -D exp -t 4 -c 100 -d 120 -L -s ./scripts/social-network/read-user-timeline.lua http://localhost:8080/wrk2-api/user-timeline/read -R 1000


5. mixed_worklaods_with_same_params: This workload is generated by running multiple workloads together. The command that generated the workload is:

    ./wrk -D exp -t 4 -c 100 -d 120 -L -s ./scripts/social-network/compose-post.lua http://localhost:8080/wrk2-api/post/compose -R 1000 &

    ./wrk -D exp -t 4 -c 100 -d 120 -L -s ./scripts/social-network/read-home-timeline.lua http://localhost:8080/wrk2-api/home-timeline/read -R 1000 &

    ./wrk -D exp -t 4 -c 100 -d 120 -L -s ./scripts/social-network/read-user-timeline.lua http://localhost:8080/wrk2-api/user-timeline/read -R 1000 &


5. mixed_worklaods_with_same_params_hour: This workload is generated by running multiple workloads together. The command that generated the workload is:

    ./wrk -D exp -t 10 -c 1000 -d 3600 -L -s ./scripts/social-network/compose-post.lua http://localhost:8080/wrk2-api/post/compose -R 10000 &

    ./wrk -D exp -t 10 -c 1000 -d 3600 -L -s ./scripts/social-network/read-home-timeline.lua http://localhost:8080/wrk2-api/home-timeline/read -R 10000 &

    ./wrk -D exp -t 10 -c 1000 -d 3600 -L -s ./scripts/social-network/read-user-timeline.lua http://localhost:8080/wrk2-api/user-timeline/read -R 10000 &
